Welcome to the project of Group3

Thank you for visiting this website. This is a student project for the course Learning Analytics in University of Duisburg-Essen.
The goals of project are as below
1.Visualize the relationship between some feature data and student performance. Then analyze which feature have a greater impact on student performance
2.Learn and analyze student feature sets through the machine learning algorithms, and try to increase or decrease features based on the previously visualized graphics to improve the accuracy of the model's prediction of student performance.
3.Through prediction, understand the characteristics of learners and provide relevant suggestions to learners to improve their learning processes and activities
